SOCKS4/4a/5 Server
==================

Server Supports:
 - IP4
 - domain names
 - Secure SSL

Server Not Supported:
 - IP6

Client Supports:
 - IP4
 - Secure SSL

Client Not Supported:
 - IP6
 - domain names (mostly connecting to IP's behind the firewall)

WebSite: http://jfsocks.sourceforge.net

Source : http://javaforce.sourceforge.net
