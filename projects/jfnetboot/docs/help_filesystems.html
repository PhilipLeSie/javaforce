jfNetBoot

File Systems are all read-only to clients.  Any changes the client makes is stored in the clients unique location on the server.

File Systems Setup:
-------------------
 - FileSystem : UEFI x86
 - FileSystem : UIEF arm
 - FileSystem : BIOS x86

Actions:
--------
Default: Sets the default file system used for new clients that connect to the service.
Create: Creates a new empty file system, use any of the setup instructions above to complete it.
Archive: Creates an archive of a file system.  This can be done at any time.
