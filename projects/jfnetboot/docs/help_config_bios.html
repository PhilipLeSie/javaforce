jfNetBoot

Configuring old BIOS (non UEFI) PC for networking booting:
----------------------------------------------------------
jfNetBoot does support BIOS-PC systems but upgrading to UEFI is an option.
Download latest iso release from github.com/CloverHackyColor/CloverBootloader
Burn to CD or USB drive, boot from it and install.
This will upgrade your legacy system to UEFI.
Network support is not guaranteed.
